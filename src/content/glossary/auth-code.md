Also called the EPP code (from the Extensible Provisioning Protocol the registry uses), transfer code, or authorization/authorisation key. The names are interchangeable; the value is the same per-domain secret. Treat it like a password while you hold it; once a transfer completes, the code is consumed and any future transfer requires a fresh one.
