**Microsoft 365** is the centre of gravity for most SMB clients. An MSP tech is in M365 admin centres every day: provisioning users, fixing email flow, troubleshooting Teams, managing OneDrive shares, applying conditional access.

The administrative surface is split across multiple consoles:

- **Microsoft 365 admin center**, users, licenses, basic settings
- **Exchange admin center**, mail flow, mailbox-level settings
- **SharePoint admin center**, sites, sharing policies
- **Teams admin center**, meeting policies, calling plans
- **Microsoft Entra (formerly Azure AD)**, identity, groups, conditional access, app registrations
- **Microsoft Defender / Purview**, security and compliance

Modern best practice for MSPs is to use **GDAP** (Granular Delegated Admin Privileges) rather than legacy DAP for accessing customer tenants.
